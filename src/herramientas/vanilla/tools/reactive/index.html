Señales Reactivas

info Qué es

Herramienta de agilidad multimodal. Combina señales visuales (colores) y auditivas (pitido) para provocar reacciones motoras específicas. Cada estímulo tiene una acción asignada que el usuario debe memorizar y ejecutar.

play_circle Cómo usar

  1. Edita las acciones en la leyenda inferior: asigna movimientos a cada señal.
  2. Pulsa INICIAR. Las primeras rondas muestran la acción (aprendizaje).
  3. Después, solo aparece la señal y el usuario debe recordar qué hacer.
  4. Hay 4 estímulos: 2 sonidos (agudo y grave) + 2 colores (turquesa y dorado).

lightbulb Consejos

  • Empieza con 2 señales y añade más cuando dominen las primeras.
  • Cambia las acciones entre rondas para forzar reaprendizaje.
  • Combina con desplazamientos: el usuario se mueve mientras espera la señal.
  • Proyecta en pantalla grande para uso grupal.

fitness_center Cuándo aplicar

  • Entrenamiento reactivo multicanal (visual + auditivo).
  • Deportes de combate: reaccionar a señales del oponente.
  • Sesiones grupales de alta intensidad con componente cognitivo.
  • Prevención de caídas: reaccionar a estímulos inesperados.
