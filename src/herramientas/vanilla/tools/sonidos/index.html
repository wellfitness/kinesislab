Sonidos Audiomotores

info Qué es

Entrenamiento de reacción auditiva. Asocia sonidos de distintas frecuencias (grave, agudo, medio) a acciones motoras específicas. Trabaja la discriminación auditiva y la velocidad de procesamiento sin estímulo visual.

play_circle Cómo usar

  1. Configura las acciones: edita los campos de texto para asignar el movimiento que quieras a cada sonido.
  2. Pulsa INICIAR. Las primeras rondas muestran la acción (fase de aprendizaje).
  3. Tras el aprendizaje, el texto desaparece y el usuario debe recordar las asociaciones.
  4. Ajusta la cadencia para subir o bajar dificultad.

lightbulb Consejos

  • Personaliza las acciones según tu sesión: saltos, sentadillas, giros, palmadas...
  • Ideal para trabajar sin mirar pantalla — el usuario puede estar de espaldas.
  • En grupos: cada persona puede tener una acción diferente al mismo sonido.
  • Cambia las acciones entre series para añadir carga cognitiva extra.

fitness_center Cuándo aplicar

  • Doble tarea: ejecutar ejercicio principal mientras se reacciona al sonido.
  • Trabajo con personas con baja visión o entrenamiento sensorial auditivo.
  • Deportes donde la señal auditiva es clave (silbatos, indicaciones verbales).
  • Fases de vuelta a la calma con reactividad baja.
