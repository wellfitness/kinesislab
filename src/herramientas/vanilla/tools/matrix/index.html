Matriz Visoespacial

info Qué es

Entrenamiento de memoria visoespacial basado en el test de Corsi. Se iluminan celdas en una cuadrícula; debes recordar cuáles se activaron y tocarlas. El modo orden avanzado exige recordar también la secuencia exacta.

play_circle Cómo usar

  1. Elige tamaño de cuadrícula (3x3, 4x4 o 5x5) y cadencia.
  2. Pulsa INICIAR. Se iluminan varias celdas momentáneamente.
  3. Cuando pida respuesta, toca las celdas que recuerdes.
  4. Modo orden: las celdas muestran números; debes tocarlas en ese orden.

lightbulb Consejos

  • Empieza en 3x3 sin modo orden para establecer línea base.
  • El modo orden multiplica la dificultad — reservar para usuarios avanzados.
  • Combina con equilibrio o sentadillas isométricas durante la fase de memorización.
  • 5x5 con modo orden es un reto extremo, incluso para jóvenes deportistas.

fitness_center Cuándo aplicar

  • Evaluación de span visoespacial (análogo al Corsi Block).
  • Doble tarea con carga espacial: ideal para deportes de posición.
  • Rehabilitación cognitiva tras ictus o TCE.
  • Estimulación cognitiva en adultos mayores.
