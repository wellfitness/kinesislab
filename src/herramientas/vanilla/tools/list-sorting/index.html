Listas de Objetos

info Qué es

Adaptación del List Sorting Working Memory Test (NIH Toolbox). Se presentan objetos de una o dos categorías (animales y alimentos) y el usuario debe ordenarlos mentalmente por tamaño. Trabaja retención temporal, reorganización y categorización.

trending_up Niveles

  • 1 Lista: Objetos de una sola categoría (solo animales o solo alimentos). Ordénalos de menor a mayor tamaño.
  • 2 Listas: Objetos mezclados de ambas categorías. Primero ordena los alimentos por tamaño, luego los animales por tamaño.
  • Inverso: Como 2 Listas, pero ordena de mayor a menor tamaño.

play_circle Cómo usar

  1. Elige nivel, cantidad de ítems (3-5) y cadencia.
  2. Pulsa INICIAR. Los objetos aparecen uno a uno.
  3. Cuando termine la presentación, toca los objetos en el orden correcto.
  4. El color indica la categoría: turquesa = animal, amarillo = alimento.

lightbulb Consejos

  • Empieza con 1 Lista y 3 ítems para establecer línea base.
  • 2 Listas añade categorización a la ordenación: doble carga cognitiva.
  • El modo Inverso es el más exigente: categorización + orden invertido.
  • Combina con marcha, equilibrio o sentadillas isométricas durante la memorización.

fitness_center Cuándo aplicar

  • Evaluación de memoria de trabajo (análogo al LSWM del NIH Toolbox).
  • Doble tarea con carga de memoria durante ejercicio aeróbico.
  • Adultos mayores: estimulación de retención y reorganización.
  • Rehabilitación cognitiva: span de memoria y categorización semántica.
