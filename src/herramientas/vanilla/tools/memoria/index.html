Memoria Visual

info Qué es

Clásico juego de parejas con cartas ocultas. Voltea dos cartas por turno e intenta encontrar todas las parejas. Trabaja memoria espacial, atención sostenida y estrategia de búsqueda. Ideal para trabajo en parejas.

play_circle Cómo usar

  1. Elige nivel: fácil (3 parejas), medio (6) o difícil (10).
  2. Pulsa COMENZAR. Las cartas se barajan.
  3. Toca una carta para voltearla. Luego toca otra.
  4. Si coinciden, quedan descubiertas. Si no, se ocultan de nuevo.

lightbulb Consejos

  • En parejas: cada persona voltea una carta, deben colaborar para recordar.
  • Añade regla motora: “entre cada turno, haz 3 sentadillas”.
  • Usa el número de intentos como métrica de progresión.
  • Nivel difícil (10 parejas) es un reto real incluso sin tarea motora.

fitness_center Cuándo aplicar

  • Trabajo cooperativo en parejas o pequeños grupos.
  • Pausa activa entre series de fuerza.
  • Estimulación cognitiva accesible para cualquier edad.
  • Vuelta a la calma con componente lúdico y social.
