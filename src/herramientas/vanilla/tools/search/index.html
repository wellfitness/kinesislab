Búsqueda Visual

info Qué es

Entrena la atención selectiva visuoespacial. Localiza un elemento distinto (letra, símbolo) entre una cuadrícula de distractores. Mide velocidad de rastreo visual y capacidad de filtrar información irrelevante.

play_circle Cómo usar

  1. Ajusta la cadencia (tiempo para encontrar el objetivo).
  2. Pulsa INICIAR. Aparece una cuadrícula con un intruso entre distractores.
  3. Toca el elemento diferente lo más rápido posible.
  4. Se registran aciertos, errores y tiempo de reacción.

lightbulb Consejos

  • Empieza con cadencia lenta para que el usuario entienda el patrón.
  • Combina con ejercicio aeróbico: buscar el intruso mientras se pedalea o camina.
  • En grupo: proyecta y que compitan por velocidad de detección.
  • Útil para evaluar la fatiga visual durante sesiones largas.

fitness_center Cuándo aplicar

  • Deportes con alta demanda visual: deportes de raqueta, porteros.
  • Doble tarea con carga visuoespacial sobre base motora.
  • Rehabilitación de atención tras lesión cerebral.
  • Calentamiento cognitivo antes de actividades de precisión.
