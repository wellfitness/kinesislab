N-Back Visual

info Qué es

Gold standard del entrenamiento de memoria de trabajo. Debes detectar cuándo el estímulo actual coincide con el de N posiciones atrás. En nivel difícil, trabaja con dos dimensiones independientes (forma y color) — basta que UNA coincida.

play_circle Cómo usar

  1. Elige nivel: fácil (1-Back, solo forma), medio (1-Back, forma+color), difícil (2-Back dual).
  2. Pulsa INICIAR. Observa la secuencia de estímulos.
  3. Cuando el actual coincida con el de N turnos atrás, pulsa COINCIDE.
  4. El historial visual inferior ayuda a recordar la secuencia.

lightbulb Consejos

  • El 1-Back fácil es accesible para casi cualquier población.
  • Combina con caminata, bicicleta estática o equilibrio para doble tarea.
  • El 2-Back dual es muy exigente: úsalo solo con usuarios avanzados.
  • Monitoriza omisiones vs falsos positivos para ajustar dificultad.

fitness_center Cuándo aplicar

  • Entrenamiento específico de memoria de trabajo (evidencia científica sólida).
  • Doble tarea cognitivo-motora de alta demanda.
  • Evaluación de capacidad cognitiva pre/post sesión.
  • Población mayor: prevención de deterioro cognitivo.
