De Menor a Mayor

info Qué es

Ejercicio de memoria de trabajo y ordenación mental. Aparecen 3 números (o operaciones) y el usuario debe ordenarlos de menor a mayor en voz alta. Trabaja la manipulación mental de información numérica.

play_circle Cómo usar

  1. Elige nivel: números simples, sumas/restas o multiplicaciones.
  2. Ajusta la cadencia según la población.
  3. Pulsa INICIAR. Aparecen 3 valores.
  4. El usuario los dice en voz alta ordenados de menor a mayor.

lightbulb Consejos

  • Números simples es accesible para casi cualquier población.
  • Las operaciones añaden carga de cálculo: el usuario debe resolver Y ordenar.
  • No hay botón de respuesta — el usuario verbaliza. Tú validas.
  • En grupo: “quien lo diga primero” añade competitividad.

fitness_center Cuándo aplicar

  • Doble tarea con carga aritmética durante ejercicio aeróbico.
  • Población mayor: mantenimiento de habilidades numéricas.
  • Deportistas: entrenamiento cognitivo bajo fatiga física.
  • Sesiones grupales: proyectar y que todos respondan simultáneamente.
