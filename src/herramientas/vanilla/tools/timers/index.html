Temporizador Clínico

info Qué es

Reloj de alto contraste diseñado para ser visible desde lejos en la sala de entrenamiento. Marca tiempo de trabajo de forma clara y legible. Herramienta de soporte esencial para cualquier formato de sesión.

play_circle Cómo usar

  1. Pulsa START para iniciar el cronómetro.
  2. Pulsa PAUSE para pausar y reanudar.
  3. Pulsa RESET para volver a cero.
  4. El número grande es visible a distancia en TV o tablet.

lightbulb Consejos

  • Proyecta en pantalla grande para que todos vean el tiempo.
  • Úsalo como base para AMRAP (“tantas rondas como puedas en X minutos”).
  • Combina con herramientas cognitivas: el timer corre mientras hacen tareas.
  • Ideal para pausas medidas entre series.

fitness_center Cuándo aplicar

  • Cualquier formato temporizado: AMRAP, EMOM, Tabata, intervalos.
  • Pausas controladas entre series de fuerza.
  • Tests cronometrados (TUG, 30s sit-to-stand, etc.).
  • Base temporal para cualquier otra herramienta cognitiva.
