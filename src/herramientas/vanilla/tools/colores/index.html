Colores Reactivos

info Qué es

Tarea de reacción cromática. La pantalla cambia de color y el paciente debe ejecutar el movimiento asignado a cada color. Entrena la asociación estímulo-respuesta y la velocidad de procesamiento visual.

play_circle Cómo usar

  1. Asigna un movimiento a cada color (ej: rojo = sentadilla, azul = salto).
  2. Selecciona la cadencia según el nivel del grupo.
  3. Pulsa INICIAR. La pantalla cambia de color aleatoriamente.
  4. El paciente ejecuta el movimiento correspondiente lo más rápido posible.

lightbulb Consejos

  • Empieza con 2-3 colores y añade más a medida que dominan.
  • Escribe las asignaciones en una pizarra visible para todos.
  • Cambia las asignaciones entre series para evitar automatización.
  • Combina con equilibrio sobre superficie inestable.

fitness_center Cuándo aplicar

  • Calentamiento dinámico con componente cognitivo.
  • Trabajo de doble tarea en adultos mayores.
  • Entrenamiento reactivo en deportes de equipo.
  • Rehabilitación: discriminación visual y asociación motora.
