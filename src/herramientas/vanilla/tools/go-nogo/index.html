Go / No-Go

info Qué es

Test clásico de control inhibitorio. Entrena la capacidad de ejecutar una respuesta rápida (Go) y frenarla cuando aparece una señal de stop (No-Go). Fundamental para la toma de decisiones bajo presión temporal.

play_circle Cómo usar

  1. Elige nivel: básico, con distractores o invertido.
  2. Ajusta la cadencia según el grupo.
  3. Pulsa INICIAR. Aparece un círculo verde (toca) o rojo (no toques).
  4. El modo invertido invierte las reglas: rojo = toca, verde = frena.

lightbulb Consejos

  • Empieza en básico para establecer una línea base de tiempo de reacción.
  • Combina con desplazamientos laterales o sentadillas mientras esperan el estímulo.
  • El modo invertido es excelente para romper patrones automáticos.
  • Monitoriza los “falsos positivos” como indicador de impulsividad.

fitness_center Cuándo aplicar

  • Calentamiento cognitivo con alta activación.
  • Trabajo de control de impulsos en deportes de equipo.
  • Evaluación rápida de fatiga cognitiva (si los falsos suben, hay fatiga).
  • Rehabilitación: pacientes con déficit de atención.
